# 📧 متن دعوت‌نامه برای دوستان

کپی و اصلاح کنید و برای دوستانتان بفرستید:

---

## **پیام 1: دعوت عمومی**

```
سلام [نام دوست]! 👋

امیدوارم حالت خوب باشه! 😊

من یک پروژه جدید شروع کردم برای دستیابی به نشان‌های GitHub و 
می‌خواستم ببینی که می‌تونی کمک کنی؟

ریپوزیتوری: https://github.com/ramincsy/Auto

می‌تونی یکی از این کار‌ها انجام بدی:

✅ کدم رو بررسی کنی (Code Review)
✅ PR رو ادغام کنی (Merge)
✅ با هم کدی بنویسیم (Co-Author)

اگر علاقه داری، این لینک رو ببین:
👉 https://github.com/ramincsy/Auto/blob/main/FRIENDS_GUIDE.md

چقدر وقتت هست؟ 💪
```

---

## **پیام 2: دعوت برای Code Review**

```
سلام [نام دوست]! 

فقط یه درخواست: می‌تونی یک نگاه به کد من بیندازی و یه کامنت مثبت بنویسی؟

ریپوزیتوری: https://github.com/ramincsy/Auto/pulls

مراحل ساده‌ای هست:
1. لینک بالا رو باز کن
2. یه PR رو انتخاب کن
3. روی "Review changes" کلیک کن
4. "Approve" رو انتخاب کن
5. کامنت بنویسی

5 بار دیگری این کار انجام بدی، من یک نشان خوبی می‌گیرم! 🏆

منتظرم 😊
```

---

## **پیام 3: دعوت برای Co-Authored Commit**

```
سلام [نام دوست]!

می‌خوای یه چیز جالب کار کنیم؟ 

بیا با هم یک commit بنویسیم! این متفاوته و هر دو‌مان نشان می‌گیریم.

ریپوزیتوری: https://github.com/ramincsy/Auto

مراحل:
1. Repository رو clone کن
2. یه شاخه جدید بساز
3. یک فایل رو تغییر بده
4. این دستور رو اجرا کن:

   git commit -m "feat: improve code together
   
   Co-authored-by: [Your Name] <your@email.com>"

5. Push کن و PR بساز!

بیا شروع کنیم! 🚀
```

---

## **پیام 4: دعوت برای Merge PR**

```
سلام [نام دوست]!

یه کار ساده برای کمک:
می‌تونی یه PR رو برای من ادغام کنی؟

ریپوزیتوری: https://github.com/ramincsy/Auto/pulls

فقط:
1. یک PR رو انتخاب کن
2. روی "Merge pull request" کلیک کن
3. تأیید کن!

همین! 2 بار این کار کنی، من یک نشان جدید می‌گیرم! 🐋

ممنون 🙏
```

## **نکات مهم:**

1. **اصلاح شخصی:** نام دوستات رو تعویض کنید
2. **زمان‌بندی:** بهترین وقت برای فرستادن: صبح یا شام
3. **نپیگیری:** اگر جواب ندادند، 1-2 روز صبر کنید سپس یادآوری کنید
4. **سپاسگزاری:** همیشه تشکر کنید!
